# Task 4 – Cancer Data Analysis

This project analyzes a cancer dataset to identify patterns and relationships between medical features.

## Objective
To explore and visualize cancer-related data and identify correlations among features using Python.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Analysis
- Data cleaning and preprocessing
- Distribution analysis of numerical features
- Correlation heatmap to understand feature relationships

## Outcome
The analysis highlights strong correlations between several tumor characteristics, helping in understanding factors associated with cancer diagnosis.

## Dataset
Source: Kaggle – Cancer Dataset
